com o projeto aberto vc tem acesso aos arquivos o main.py é o principal se quiser executar o aplicativo na sua máquina  vá para o terminal do vscode e rode o seguinte comando python -m streamlit run main.py
é preciso ter o python instalado na sua máquina, para instalar o python direto do vscode ao lado tem alguns icones selecione o icone de extensão e instale o python ou baixe o python na sua máquina e instale.
O pacote já deve estar com o streamlint mas caso reclame vá no terminal do vscode e rode !pip install streamlint. Se houver algum problema entre em contato para resolvermos. Em outros editores Jupyter, anaconda
